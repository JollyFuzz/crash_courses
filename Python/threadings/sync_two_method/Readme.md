# Синхронизация многопоточности
## Постановка задачи

Необходимо разобраться с синхронизацией многопоточного доступа в Python на примере класса, внутри которого есть метод, выполняющий длительную операцию. Для наглядности вместо реального сетевого запроса должна использоваться имитация асинхронного ожидания.
